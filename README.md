# Arotech Digital Store

Arotech Digital Store is a catalog-driven electronics website built for showcasing and managing tech products such as phones, laptops, inverters, and accessories.

This repository is a **case study and project overview**.  
The full source code is private because it is a live commercial project.

Live site: https://arotech.com.ng

---

## Project Overview

The goal of this project was to build a clean and reliable product catalog that allows customers to:

- Browse electronics by category
- View detailed product information
- Contact the business easily for inquiries
- Access the site on both mobile and desktop devices

The focus was on clarity, performance, and simple navigation.

---

## Key Features

- Product catalog with categories
- Product detail pages with images and descriptions
- Mobile-friendly layout
- Simple admin-managed content
- Fast load times and clean UI
